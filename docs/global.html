(constant) rATag :string

匹配 html 的 a 标签：

  1. /...../ 为正则表达式的界定符，/i 的 i 是标志位，表示不区分大小写
  2. a 标签的起始分隔符直接匹配
  3. (?: ...) 非捕获组，匹配内容但不捕获
  4. \s+? 匹配至少一个空白字符，\s 匹配任意空白字符，+? 懒惰量词，尽可能少地匹配
  5. |\s+?[^<>]+\s+? 或者匹配一些空白字符之后的非尖括号内容（至少一个）以及后面的空白字符，[^<>] 表示非尖括号内容
  6. href=["'] 匹配 href，["'] 表示单引号或者双引号
  7. (?:#) 非捕获组捕获 #
  8. ([^<>"']+) 捕获非尖括号和单双引号外的字符至少一个，+ 表示至少一次或以上
  9. ["'] 匹配结束引号
  10. [^<>]* 匹配非尖括号外字符零次或多次，* 代表零次及以上

(constant) rDlSyntax

匹配以换行符
和冒号 : 为分隔的键值对结构

  1. /..../ 为正则表达式的界定符
  2. (?:^|\s) 表示一个非捕获组，匹配字符串 (\S.+) 的开头 (^) ，或一个空白字符 (\s)
  3. (\S.+) 表示捕获以非空白字符 (\S) 开头，后跟任意字符 (.) 至少一次的字符串
  4. <br>: 表示直接匹配字符串
    和冒号 :
  5. \s+(\S.+) 表示匹配一个或多个空白字符 (\s+) 后接以非空白字符开头的字符串 (\S.+) ，长度至少为2

生成规范化的 ID 字符串，作为 html 元素的 ID 或者锚点链接：
